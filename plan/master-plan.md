## 一、真实战情

### 基线分数（2026-04-10 诊断）

| 科目 | 基线 | 目标 | 差距 | 现状描述 |
|------|------|------|------|---------|
| 阅读 | 21 | 27 | +6 | 理解力在，细节和推理丢分 |
| 听力 | 15 | 26 | +11 | 12年未系统听英语，预计很差 |
| 口语 | 16 | 23 | +7 | 能说但语法错误密集，用词单一 |
| 写作 | 10 | 25 | +15 | 几乎无法产出完整段落 |
| **总计** | **~62** | **100+** | **+38** | — |

### 必须面对的事实

1. **+38分的差距，60小时的时间**，这是一场硬仗
2. 写作从10到25，等于从零建起，这是最大的工程
3. 听力+11的差距，光靠每天2小时不够，需要碎片时间持续灌入
4. 好消息：你的底层能力（逻辑、结构化思维、语法知识储备）都在，需要的是**重建输出习惯**

### 调整后的核心策略：重建输出 + 听力浸泡

之前的策略是"除锈"，现在要改成**"重建"**：
- 写作：不是"练几篇"，是要从"写不出来"练到"能稳定产出300字议论文"
- 口语：不是"修一修语法"，是要建立**可靠的输出模板**让你在压力下不崩
- 听力：2小时内练不够，必须加碎片时间（通勤、做饭、睡前都听英语）
- 阅读：反而是最省时的，底子好，刷题就能涨

## 二、时间分配策略

### 每日2小时核心时间分配

| 科目 | 前15天 | 后15天 | 为什么 |
|------|--------|--------|--------|
| 写作 | **50min（42%）** | 35min（29%） | 前期重建输出能力，后期转为保持 |
| 听力 | 30min（25%） | 35min（29%） | 精听训练，配合碎片时间 |
| 口语 | 25min（21%） | 25min（21%） | 稳定训练，模板内化 |
| 阅读 | 15min（12%） | 25min（21%） | 前期少练，后期刷题冲分 |

### 碎片时间（额外1-1.5小时/天）

这不是"建议"，是**必须**：
- 🎧 通勤/走路：听 TPO lecture（不用做题，只练耳朵）
- 🎧 做饭/家务：听英语播客（推荐 Scientific American 60s）
- 📱 睡前15min：背当日词汇（用手机复习）
- 这部分不计入2小时，但对听力提分至关重要

## 三、五阶段作战计划

### 第一阶段：重建期（Day 1-7）⏱️ 14小时

> 关键词：从"完全不会"到"能写出来、能说完整"

**写作重建路线（每天50min）：**

| 天 | 写作任务 | 目标 |
|----|---------|------|
| D1 | ✅ 诊断完成 | 确认写作基线=几乎零 |
| D2 | 学习托福写作结构：开头-理由1-理由2-结论 | 脑子里有框架 |
| D3 | AI给骨架，你填充每段的3-4句话 | 从填空开始写 |
| D4 | AI给开头，你续写完整篇 | 能写150词 |
| D5 | 独立写一篇，AI逐句批改，你改写一遍 | 能写200词 |
| D6 | 再写一篇，重点练理由段的展开 | 能写250词 |
| D7 | 限时30min写一篇完整的，自评+AI评 | 能写300词，有结构 |

**口语重建路线（每天25min）：**
- D2-D3：学习 Task 1-4 的答题模板
- D4-D7：每天练2题，录音→AI批改→重说

**听力启动（每天30min核心 + 碎片时间）：**
- D2-D4：0.75x慢速听 TPO conversation，逐句跟读
- D5-D7：1.0x正常速度，练记笔记

**阅读（每天15min）：**
- D2-D7：每天做半篇TPO阅读（5题），限时7分钟

**阶段目标**：写作能独立完成300词文章，口语能说满45秒，听力能跟上conversation

### 第二阶段：提速期（Day 8-14）⏱️ 14小时

> 关键词：从"能做"到"做得快、做得准"

**写作（50min/天）：**
- 每2天写1篇完整独立写作（30min限时）
- AI批改后，**背诵AI改写版的关键句型**（这是捷径）
- 开始练综合写作（听+读+写）

**口语（25min/天）：**
- Task 1：每天3题，追求流畅度
- Task 2-4：开始练综合口语（读+听+说）
- 重点：消灭高频语法错误（动名词、主语it、比较级）

**听力（30min核心 + 碎片）：**
- 从 conversation 升级到 lecture
- 精听法：听→记笔记→做题→对原文→重听
- 碎片时间：每天至少额外听40min

**阅读（15min/天）：**
- 每天1篇完整TPO阅读（10题，限时18min）
- 重点练：细节题和推理题（你的丢分点）

**阶段目标**：写作能在30min内写出结构完整的300词+文章，口语Task 1能拿20+

## 四、写作专项突破方案（最大短板）

写作从10到25是全计划最大的工程。详细拆解：

### 第一周：建立输出能力
```
你现在 → 看到题目脑子有想法但写不出英文
目标   → 能用简单句把想法写出来，哪怕语法不完美
方法   → AI搭骨架你填肉 → AI给开头你续写 → 你独立写AI改
```

### 第二周：提升质量和速度
```
你现在 → 能写出来但慢、错多、展不开
目标   → 30分钟300词，结构清晰，语法错误<5处
方法   → 限时练习 → 背诵AI改写版好句子 → 反复写同类题
```

### 第三周：应试优化
```
你现在 → 能写完但分不高
目标   → 稳定23-25分
方法   → 综合写作模板化 → 独立写作万能理由库 → 高分句型套用
```

### 第四周：保持
```
每天写1篇保持手感，不再加新技巧
```

### 写作提分的三个捷径

1. **背改写版**：每次AI改完你的文章，把AI版本的好句子背下来，下次直接用
2. **万能理由库**：准备5组万能理由（效率、健康、人际、经济、个人成长），任何题都能套
3. **开头结尾模板**：固定句式，省时间留给正文

## 八、诚实评估

30天从62到100+是一个**极度激进**的目标。正常备考节奏，这个跨度需要3-6个月。

我们在赌三件事：
1. 你的六级600底子还在，只是被封印了，能快速唤醒
2. AI辅助能把传统备考的效率提升2-3倍
3. 你能坚持每天2小时核心+1小时碎片，一天不断

如果这三个条件都成立，100+有可能。如果中途发现某个条件不成立，我会诚实告诉你，并建议调整目标或时间线。

**这个实验的价值不只是分数——而是验证AI能把学习效率推到什么极限。**
