# 项目5：TOEFL 100+ 三十天冲刺

> 一个用 AI 全程辅助、从零到100+的托福备考实验
> 全过程公开，所有对话与产出均可复用

## 项目双重目标

1. **个人目标**：30天内托福考到100+
2. **系统目标**：打造一套可复用的「AI辅助托福备考系统」，供后来者直接使用

## 备考者画像
- CET-6 600+（2014年裸考），底子好但12年锈蚀
- 每天投入2小时，总学时约60小时
- 全程使用 Claude Code 作为AI教练

## 如何使用这个系统

如果你也想用AI辅助备考托福：
1. Fork 这个项目结构
2. 完成 Day 1 的基线模考
3. 把分数告诉 Claude，它会自动调整 master-plan
4. 每天开始时说「托福今日计划」，获取当日任务
5. 完成后反馈结果，系统自动追踪进度
