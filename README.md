## 这是什么

一个人，一个 AI，30 天，冲刺托福 100+。

这不是事后总结的经验帖，而是一场**正在发生的公开实验**——从第一天建计划到最后一天走进考场，所有的对话、计划、练习、批改、踩坑、调整，全部实时同步在这个仓库里。

你看到的不是"我考完了来分享方法论"，而是"我正在考，你可以看着我怎么用 AI 一步步搞定它"。

## 为什么值得关注

- **全过程透明** — 每一天做了什么、AI 说了什么、哪里有效哪里踩坑，全部公开
- **真实可验证** — 有基线分数、有模考记录、有最终成绩，不是编故事
- **可直接复用** — 这套系统你 fork 下来，换上你的基线分数，就能跑

## AI 在这里扮演什么角色

不是"帮你搜搜资料"那种 AI，而是**全流程深度介入**：

| 环节 | AI 做什么 |
|------|----------|
| **规划** | 根据基线分数自动生成30天计划，每天动态调整 |
| **阅读训练** | 出题 → 限时 → 批改 → 逐题讲解 → 提取生词 |
| **听力训练** | 推荐材料 → 精听指导 → 笔记法训练 → 分析错误模式 |
| **口语训练** | 出题 → 给范文 → 逐句纠正 → 评分反馈 |
| **写作训练** | 出题 → 逐段批改 → 改写示范 → 积累句型 |
| **词汇管理** | 从练习中自动提取 → 间隔复习 → 语境测验 |
| **模考分析** | 自动分析错题规律 → 定位薄弱点 → 调整后续计划 |

## Fork 你自己的 AI 托福之旅

想用同样的方式备考？

1. **Fork** 这个仓库
2. 做一套 TPO 模考，记录你的基线分数
3. 把你的分数和背景告诉 Claude，让它调整 `plan/master-plan.md`
4. 每天对 Claude 说 **"托福今日计划"**，它会自动生成当天任务
5. 完成后反馈结果，AI 自动追踪进度、调整计划
6. 重复步骤 4-5，直到考试日
